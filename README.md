# Stack Overflow Clone

## Project Overview
This project was initially developed as a training project to replicate the core functionalities of Stack Overflow using the MERN stack.

## Project Evolution
During my internship, I extended this project by implementing advanced features and real-world functionalities.

## Internship Features Added

- Social media system (posts, likes, comments, sharing)
- Friend-based posting limits
- Subscription plans with payment integration
- Reward points system with transfer functionality
- Forgot password with daily restriction and password generator
- Multi-language support with OTP verification
- Login tracking (browser, OS, device, IP)
- Device-based and time-based authentication

## Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: JWT
- Payment Integration: Razorpay / Stripe
